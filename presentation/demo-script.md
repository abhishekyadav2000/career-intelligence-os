# Demo Script

## 2-Minute Version

> "Career Intelligence OS is a sponsor-aware decision system for enterprise technology roles in DFW. It ingests company, job, and contact data, scores role fit across six categories, detects posting noise, and recommends whether to apply, network, research, or skip. Let me show you the dashboard."

**Show:** Overview KPIs → Company Ranking top 5 → one Role Fit drill-down → one Recommendation.

**Close:** "This demonstrates Python data engineering, scoring logic, risk-aware sponsorship signals, and business communication — all in one portfolio project."

---

## 5-Minute Version

1. **Problem (30s)** — DFW has 50+ tech employers; decisions are fragmented.
2. **Architecture (30s)** — CSV → validate → score → recommend → dashboard.
3. **Overview tab (45s)** — 50 companies, 117 jobs, avg fit score, gap matrix.
4. **Company Ranking (45s)** — Tier-1 employers, priority scores, pipeline depth.
5. **Role Fit (60s)** — Six categories, drill into JPMorgan Chase role, show category chart.
6. **Sponsorship Signal (45s)** — Disclaimer, signal scores, not legal advice.
7. **Recommendations (45s)** — apply now vs network first with rationale.
8. **Close (30s)** — Export CSV, SQL demo available, prompt templates for future AI agents.

---

## 10-Minute Version

Add to 5-minute flow:

9. **Networking Map (60s)** — Outreach angle for recruiter vs hiring manager.

11. **Export & SQL (60s)** — Download scores CSV, run tier-1 pipeline query.
12. **Business case (60s)** — Case study, risk register, gap matrix connection.
13. **Q&A prep (60s)** — "Why not use LinkedIn?" → intentional scope; "Is sponsorship guaranteed?" → disclaimer; "What's next?" → LLM agents, live feeds.
