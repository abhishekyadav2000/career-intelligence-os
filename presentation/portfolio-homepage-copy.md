# Portfolio Homepage Copy

## Project Title
Career Intelligence OS

## Tagline
Sponsor-aware career intelligence system for enterprise technology roles.

## One-Liner
An AI-enabled enterprise decision system for role-fit scoring, talent intelligence, outreach strategy, and interview readiness — built with Python, Streamlit, and structured DFW market data.

## Description
Career Intelligence OS transforms fragmented job search data into actionable intelligence. It scores 117 enterprise technology roles across 50 DFW employers using six decision categories — technical fit, business fit, sponsorship signal, DFW relevance, networking opportunity, and noise risk — then recommends whether to apply, network, research, or skip.

The system demonstrates data engineering (CSV validation, normalization), analytics (weighted scoring, KPI dashboard), business analysis (requirements, risk register), and security thinking (sponsorship disclaimers, ghost job detection).

## Tech Stack
Python · pandas · Streamlit · SQLite · scikit-learn · Plotly

## Key Metrics
- 50 companies · 117 jobs · 75 contacts
- 6 scoring categories · 4 recommendation actions
- 8 dashboard tabs · CSV export · SQL analytics demo
