# Recruiter Version

## Elevator Pitch

I built Career Intelligence OS — a decision system that scores enterprise technology roles against my profile, prioritizes 50 DFW employers, and generates outreach and interview prep. It's not a job board; it's an analytics pipeline that shows how I think about data-driven career decisions.

## What to Highlight

- **Data engineering:** CSV ingestion with schema validation and normalization
- **Analytics:** Six-category weighted scoring with explainable breakdown
- **Business sense:** Recommendations (apply/network/research/skip) with rationale
- **Risk awareness:** Sponsorship signals flagged as indicative, not legal certainty
- **Communication:** Rule-based outreach templates by contact type

## Key Numbers

- 50 DFW companies, 117 jobs scored, 75 contacts mapped
- 6 scoring categories, 4 recommendation actions
- 8 dashboard tabs with CSV export

## If Asked "Why Should We Interview You?"

This project shows I don't just apply randomly — I build systems to prioritize effort, interpret signals carefully, and communicate professionally. The same skills translate to technology analyst, data analyst, and cloud security analyst roles.
