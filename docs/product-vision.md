# Product Vision

## North Star

**Career Intelligence OS** — An AI-enabled enterprise decision system for role-fit scoring, talent intelligence, outreach strategy, sponsor-aware targeting, and interview readiness.

**Tagline:** Sponsor-aware career intelligence system for enterprise technology roles.

## What This Is

An enterprise decision system that helps analysts and candidates prioritize DFW technology employers, score role fit against a universal profile, interpret sponsorship signals, detect posting noise, and plan outreach — with full auditability.

## What This Is NOT (MVP Scope)

- Not a job board or application tracker
- Not a LinkedIn automation or scraping tool
- Not a legal immigration advisory system
- Not a user-authenticated SaaS product (yet)

## Universal Profile

> Enterprise Technology Analyst — AI Automation, Cloud Security & Data Analytics

## Core Capabilities

| Layer | Capability |
|-------|-----------|
| Data | CSV ingestion, schema validation, normalization |
| Scoring | Role fit, company priority, sponsorship signal, noise detection |
| Decision | Recommendations: apply / network / research / skip |
| Communication | Outreach angles, interview prep, gap analysis |
| Presentation | Streamlit dashboard, CSV export, SQL demo |
| AI (Phase 5) | Rule-based generators + prompt templates for future LLM agents |

## Future Roadmap (Deferred)

- LLM-powered company research agent
- Live job feed ingestion
- User accounts and saved searches
- Integration with DOL/USCIS public data APIs
- Mobile companion app
