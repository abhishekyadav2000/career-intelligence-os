# Interview Packet: Technical Project Coordinator

**Role Family:** Technical Project Coordinator  
**Universal Profile Fit:** Enterprise Technology Analyst — AI Automation, Cloud Security & Data Analytics  
**Typical Employers:** Deloitte, Accenture, Cognizant, Infosys, KPMG, JPMorgan Chase

---

## Role Summary

Technical project coordinators manage project timelines, track deliverables, facilitate communication between technical and business teams, and ensure projects stay on scope and schedule. In enterprise consulting and finance-tech, this role requires Agile fluency, risk tracking, and stakeholder reporting.

## Technical Questions (Prepare Answers)

1. **"How do you track project progress?"**
   → Gap matrix with P0/P1 priorities; conversation log with follow-up dates; dashboard KPIs as status metrics.

2. **"Describe your experience with Agile/Scrum."**
   → CI OS built iteratively: data layer → scoring → decision → communication → dashboard → feedback loop. Each sprint added a layer.

3. **"How do you manage scope creep?"**
   → Recommendation engine: four clear actions prevent scope ambiguity. "Research more" = needs definition before proceeding.

4. **"What project management tools have you used?"**
   → Jira (keyword in taxonomy), GitHub (version control), Streamlit (delivery), pytest (quality gates).

5. **"How do you handle a blocked dependency?"**
   → CI OS dependency order: data → scoring → recommendations. Can't generate outreach without scores. Documented in architecture.
